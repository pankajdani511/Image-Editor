# 🖼️ Image Editor

A simple and interactive **Image Editor** built using **HTML, CSS, and JavaScript**.  
This project allows users to apply various filters and adjustments to images directly in the browser.

---

## 🚀 Features

- 📤 Upload images from your device  
- 🎚️ Adjust image filters in real-time:
  - Brightness
  - Contrast
  - Exposure
  - Saturation
  - Hue Rotate
  - Blur
  - Grayscale
  - Sepia
  - Opacity
  - Invert
- 🔄 Reset all filters with one click
- 💾 Download the edited image
- 📱 Responsive design (works on desktop & mobile)

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Image processing logic and interactivity
